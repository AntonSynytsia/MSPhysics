General Properties

Joints are the most robust on exact solver or on iterative solver with 64 passes.

Constraint Type

Fixed

Hinge

Hinge joint has two modes:

  1. Friction mode: Hinge with rotational friction. Use the Friction parameter to adjust rotational friction. The Controller parameter can be used to amplify/reduce frictional strength. Actual friction is friction • controller. Disable the Rotate Back feature to use this mode.
  2. Spring mode: Hinge that rotates back to the desired starting angle. Use Stiff and Damp parameters to control spring acceleration and strength. Use Start Angle parameter to preset the desired starting angle. The Controller parameter can be used to modify the starting angle. The actual starting angle, in degrees, is start_angle • controller. Enable the Rotate Back feature to use this mode.

Motor

The maximum angular rate, in radians per second, is accel • controller / damp.

Servo

The maximum angular rate, in radians per second, is accel / damp.
